🧩 Problem: Contains Duplicate

LeetCode Contains Duplicate problem

📄 Problem Statement

Given an integer array nums, return:

👉 true if any value appears at least twice
👉 false if all elements are distinct

🧪 Example 1:
Input: nums = [1,2,3,1]  
Output: true

👉 1 appears twice

🧪 Example 2:
Input: nums = [1,2,3,4]  
Output: false

👉 All elements are unique

🧪 Example 3:
Input: nums = [1,1,1,3,3,4,3,2,4,2]  
Output: true
🔒 Constraints:
1 <= nums.length <= 10^5
-10^9 <= nums[i] <= 10^9
