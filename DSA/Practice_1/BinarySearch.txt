Given a sorted array of integers nums and an integer target, return:

👉 The index of target if it exists
👉 Otherwise return -1

🧪 Example 1:
Input: nums = [-1,0,3,5,9,12], target = 9  
Output: 4
🧪 Example 2:
Input: nums = [-1,0,3,5,9,12], target = 2  
Output: -1
🔒 Constraints:
1 <= nums.length <= 10^4
nums is sorted in ascending order
